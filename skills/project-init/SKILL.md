# project-init：项目初始化

## 触发条件

- 用户要求初始化新项目、搭建项目脚手架、创建项目结构
- 项目中不存在 README.md 或 package.json 等基础文件

## 执行流程

```text
1. 确认项目名称、类型、技术栈
2. 确认目标平台和兼容性要求
3. 列出将要创建的文件清单和目录结构
4. 等待用户确认
5. 按顺序创建文件（先配置文件，再源码文件）
6. 创建完成后输出完整项目结构树
7. 更新 README.md 和 TODO.md
```

## 输出要求

- 项目结构树
- 每个文件的用途说明
- 后续开发建议

## 禁止事项

- 不引入不必要的依赖
- 不过度设计目录结构
- 不创建用户未确认的文件
