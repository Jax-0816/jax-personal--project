# frontend-dev：前端开发

## 触发条件

- 用户要求开发前端页面、UI 组件、样式、交互逻辑
- 任务涉及 HTML、CSS、JavaScript、React、Vue 等前端技术

## 执行流程

```text
1. 理解 UI 需求和交互逻辑
2. 确认设计风格、响应式要求、浏览器兼容性
3. 列出计划修改和新建的文件
4. 说明与现有组件的复用关系
5. 等待用户确认
6. 先写结构（HTML/JSX），再写样式（CSS），最后加交互（JS）
7. 完成后在浏览器中验证
```

## 输出要求

- 新增/修改了哪些页面或组件
- 样式变更说明
- 交互逻辑说明
- 建议的浏览器验证方式

## 禁止事项

- 不引入不必要的前端框架或库
- 不破坏已有页面布局
- 不写未确认的动画效果
- 不忽略移动端适配
